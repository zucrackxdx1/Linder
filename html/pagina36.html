Formulario - input type="button"

Otro tipo de boton que podemos crear es mediante el elemento input y fijando en la propiedad type el valor "button". Este tipo de botón no se lo puede hacer que actúe como los botones de tipo submit o reset, su actividad dependerá de un programa desarrollado en JavaScript. Si quiere puede introducirse luego en este lenguaje visitando el curso JavaScript Ya. Si bien no podemos ver su funcionamiento ya que no conocemos JavaScript si podemos implementar una página que muestre este control.
