Elemento HTML pre /pre

El elemento HTML "pre" nos permite mostrar toda la información que está contenida entre las marcas de comienzo y fin tal cual como lo hemos escrito respetando inclusive los espacios en blanco y salto de líneas. Veamos como se muestra en un navegador la información contenida dentro de esta etiqueta:

Etiqueta HTML pre

 select * from editoriales
    left join libros
    on editoriales.codigo=libros.codigoeditorial;
salir
